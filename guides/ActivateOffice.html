A guide to activating Office365

Installation

  1. Open Powershell

    Open Powershell as administrator

  2. Run the command

    Type the command irm https://get.activated.win | iex then hit enter

  3. Select Ohook

    Type 2 in the new window

  4. Install Office

    If you don't have Office365 already, type 3. If you do have it installed, skip to step 7

  5. Install Office365 installer

    Click on the link as shown in the picture below

  6. Select Office365 Version

    Choose the version that suits you, for this guide, we'll use the ProPlusRetail version.

  7. Activate Office365

    After installing Office365, go back to the command prompt and type 1

  8. Enjoy!

    You should now have free access to all Office365 Products including Access, Word, Powerpoint...
